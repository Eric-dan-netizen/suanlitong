```
└─────────────────────────────────────────────────────────────┘
```

**Slide 11 讲稿（25 秒）：**
"12 个月内完成三件事：MVP 上线，做到 500 付费用户、MRR 30 万，盈亏平衡。然后触发 A 轮。我们的 A 轮触发条件写得很具体——不是'觉得差不多了就融'，是 MRR 50 万、LTV 比 CAC 大于 5 比 1、自有 GPU 验证通过，满足两条就启动。这叫里程碑式融资，2026 年的投资人只认这个。"

**投资人最可能追问的 5 个问题（提前准备）：**

1. "你怎么保证云厂商不直接抄你的体验层？" → 回答要点：大厂不会为 5% 的用户重新设计产品，就像银行不会为大学生重做手机银行
2. "200 万够吗？为什么不是 500 万？" → 回答要点：种子轮的钱只用来验证 PMF，不是用来买卡的。PMF 验证了再融 A 轮买资产
3. "你一个人，技术合伙人还没有？" → 回答要点：开源项目已经在做招聘漏斗。MVP 我一个人能写，不需要等 CTO 划第一根火柴
4. "AutoDL 已经 10000 张卡了，你凭什么？" → 回答要点：AutoDL 的主力是 4090 消费级卡，我们的主力是 A100/H100 企业级卡。两拨用户不重叠
5. "如果大模型公司自己搞算力平台？" → 回答要点：大模型公司自己的训练任务已经把 GPU 吃满，没有闲置算力可以向外卖。算力通聚合的是云厂商的闲置资源，不是单一公司的
