# ⚡ 算力通 · SuanLiTong

> **让算力像自来水一样，扭开即用。**

面向 AI 创业者和独立开发者的多云 GPU 算力聚合平台。

[![Stage](https://img.shields.io/badge/stage-seed-red)](https://github.com)
[![License](https://img.shields.io/badge/license-Proprietary-blue)](LICENSE)

---

## 🎯 核心功能

- 🔍 **多云比价引擎** — 实时对比阿里云/华为云/腾讯云 GPU 价格
- ⏱️ **按秒计费 + Spot 竞价** — 用多少付多少，闲置时段最低 3 折
- 🚀 **一键部署** — 预置 50+ AI 框架镜像，3 分钟就绪
- 🎫 **算力券代申领** — 自动匹配所在地政策补贴

## 🚀 快速开始

```bash
# 环境初始化
make setup

# 生成 Pitch Deck PPTX
make ppt

# 查看所有命令
make help
```
